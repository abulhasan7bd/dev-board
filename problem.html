Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM: getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML includes HTML content, innerText returns only visible text, and textContent returns all text including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements, improving performance and managing dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up to its parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

Elements can be created using document.createElement, added using appendChild, and removed using remove().
